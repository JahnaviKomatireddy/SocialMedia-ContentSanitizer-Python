SOCIAL MEDIA CONTENT SANITIZER
The Social Media Content Sanitizer is a Python-based project that simulates a moderation system for a school social media 
platform.It scans user posts, filters inappropriate content, extracts web links, and generates a moderation report.

FEATURES
Dynamic user input for posts
Detects and masks banned words
Extracts and stores URLs in a file
Tracks user violations using dictionary
Generates a final report

WORKING OF PROJECT
User enters multiple posts
Each post is scanned for banned words
Banned words are replaced with ***
Links are extracted and saved to a file (links_found.txt)
System tracks how many times each user is flagged
Final report is displayed

TECHNOLOGIES USED
Python (Core Concepts)
Lists, Tuples, Dictionaries
File Handling
